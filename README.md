### 示例代码

文件夹*example*包含书中的示例程序源代码，它们是基于`clang/llvm 3.8.1`编写的。只有安装了[LLVM 3.8.1](https://releases.llvm.org/download.html#3.8.1)，才能按照书中说明的方式编译它们。
